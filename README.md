## Google Sheets Image Drag and Drop

A lightweight Chrome Extension that automates the Google Sheets menu to make inserting images into cells a single-motion experience.

Google Sheets usually requires 4-5 clicks to get an image into a cell. This extension listens for a file being dragged over the browser and instantly opens the native upload dialog.

## Privacy Policy for Sheets Image Drag and Drop

1. Data Collection: This extension does not collect, store, or transmit any personal information, browsing history, or spreadsheet data.
2. Data Processing: All operations are performed locally within the user's browser. The extension merely automates UI clicks within the Google Sheets interface to assist with image uploading.
3. Third Parties: No data is ever shared with third parties. The extension does not communicate with any external servers.
4. Permissions: The extension requires access to docs.google.com/spreadsheets solely to detect file drag events and navigate the internal menu.
